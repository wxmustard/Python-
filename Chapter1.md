# 编程简介

* **Python**

  一种计算机编程语言及一套软件工具和库。

* 做什么？

  脚本、网站开发、文本处理、科学计算、教育等

|          优点           |                缺点                 |
| :-------------------: | :-------------------------------: |
| 可以最大限度地减少程序员在项目上花费的时间 | 速度比JAVA、C#、C++等语言慢，故不适合应用于开发新操作系统 |

* 工作过程

  * 获得需求
  * 编写源代码 (.py)
  * 使用Python解释器将源代码转成目标文件(.pyc)
  * 运行

  > .py：程序，源代码，代码
  >
  > .pyc：可执行代码，可执行文件，软件
