# 编写程序

* **ubuntu**环境下安装`Python`编辑器`IDLE`

  ```bash
  # 查看本机python版本
  python --version
  # 安装IDLE编辑器
  sudo apt install idle-python2.7
  ```

* 使用`IDLE`编写第一个`Python`程序

  * 打开`IDLE`

  * File -->  NEW File

  * 在新弹出的窗口输入`Python`语句

    ```python
    print('Welcome to Python')
    ```

  * File --> Save  保存成 **.py**文件

* *运行方法一*

  * Run --> Run Moudle

  * 会出现一个新的窗口返回结果

    ![](https://vgy.me/K7qFeQ.png)

* *运行方法二*

  * ```python
    python welcome.py
    ```
